# Anonimitzador Local Multi-Format

App web 100% local per anonimitzar documents sensibles (PII) per compliment RGPD.

## Funcionalitats

- **Formats**: PDF, DOCX, TXT, CSV, imatges (JPG/PNG)
- **Detecció PII**: Regex + heurística (IBAN, NIF, emails, noms...)
- **UI mòbil**: Tap sobre text per (des)anonimitzar
- **Modes**: Original (ressaltat) / Ocult (tokens `****PERSON_0001****`)
- **Export**: Markdown + JSON mapping (rastreig)

## Ús

1. Obre `index.html` al navegador (o GitHub Pages).
2. Carrega fitxers o enganxa text.
3. Clica "Processa i detecta PII".
4. Revisa entitats (taronja = pendent).
5. Exporta MD + JSON.
